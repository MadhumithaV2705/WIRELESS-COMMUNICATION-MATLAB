# 📖 About Repository

This repository contains MATLAB programs and simulations for various Wireless Communication experiments. The experiments demonstrate important concepts such as CDMA handoff, Rayleigh fading, receiver diversity, Walsh codes, and MIMO spatial multiplexing.
